# AI Task Management Portal - Frontend

## Overview

AI Task Management Portal is a React.js-based frontend application that allows users to manage tasks efficiently. Users can register, log in, create tasks, update tasks, delete tasks, and generate AI-powered task descriptions.

## Features

* User Registration
* User Login
* JWT Authentication
* Dashboard
* Create Task
* Edit Task
* Delete Task
* AI Task Generation
* Responsive User Interface

## Technologies Used

* React.js
* React Router DOM
* Axios
* CSS
* Vite
